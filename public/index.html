— SHERLOCKAPI.CO —

sherlockapi.co is a free JSON API for hand curated Sherlock Holmes quotes, cases, and characters from the public domain works of Sir Arthur Conan Doyle. Read more

Usage

Retrieve a random Sherlock Holmes quote in JSON format.

GET https://sherlockapi.co/api/v1/quotes/random

Example response:

{ "id": "q001", "text": "When you have eliminated the impossible, whatever remains, however improbable, must be the truth.", "source": "The Sign of the Four", "character": "Sherlock Holmes", "year": 1890 }

Retrieve all cases (4 novels + 56 short stories). Pass type to filter.

GET https://sherlockapi.co/api/v1/cases
GET https://sherlockapi.co/api/v1/cases?type={novel|short_story}

Retrieve the list of main characters.

GET https://sherlockapi.co/api/v1/characters

Free text search.

GET https://sherlockapi.co/api/v1/search?q={query}
